## Install and use

- Get the project code

```bash
git clone https://github.com.cnpmjs.org/dgiot/dgiot_dashboard.git
```

- Installation dependencies

```bash
cd dgiot_dashboard

yarn dgiot:install

```

- run

```bash
yarn serve
```
